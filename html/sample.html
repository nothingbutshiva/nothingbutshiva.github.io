Sample HTML Page

Welcome

This is a sample standalone HTML page. You can add any custom HTML content here. Each page in the html/ folder is served as-is with no processing.

How to Add Pages

  1. Create a new .html file in the html/ folder
  2. Use the same navigation and styling by including the Pico CSS and custom stylesheet
  3. Add a link to your new page in html/index.html
  4. Commit and push — it will be live on GitHub Pages
