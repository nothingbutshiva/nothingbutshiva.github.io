# Welcome to Nothing But Shiva

This is a sample article written in Markdown. It demonstrates how markdown files in the `markdown/` folder are automatically converted to beautiful HTML pages.

## How It Works

1. Write your content in Markdown (`.md` files) inside the `markdown/` folder
2. On each push to `main`, GitHub Actions automatically converts them to HTML
3. The generated pages appear in the `pages/` folder with full site styling
4. Readers get a clean, consistent reading experience
